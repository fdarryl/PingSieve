# PingSieve
**PingSieve** - это утилита, способная в автоматическом режиме искать рабочие VLESS конфиги для обхода ограничений. Основное предназначение - это обход белых списков в России.

## Quick Start Guide
### Step 1: Parse Configurations
1. Подключитесь к **WiFi или 2G** (интернет без ограничений)
2. Откройте PingSieve
3. Нажмите **Загрузить** чтобы получить VLESS конфиги
4. Дождитесь полной загрузки списка
### Step 2: Check Configurations
1. Переключитесь на **LTE** (где включены белые списки)
2. Нажмите кнопку **Проверить** чтобы начать проверку
3. Приложение автоматически пропингует каждый IP
4. Рабочие конфигурации появятся в разделе "Действительные"
5. Дождитесь завершения процесса
### Step 3: Use in Launcher
1. Скопируйте рабочие конфигурации из раздела "Действительные" (нажмите **Копировать в буфер обмена**)
2. Откройте любой лаунчер с поддержкой VLESS (например Nekoray, V2rayNG, Hiddify)
3. Импортируйте конфигурации в лаунчер
4. Проверьте **реальный delay/ping** для каждой конфигурации
5. Если пинг отображается — конфиг подходит
6. Используйте рабочую конфигурацию для подключения
## Installation
1. Скачайте последний APK с [релизов](https://github.com/fdarryl/VlessChecker/releases/latest)
2. Включите "Установка из неизвестных источников"
3. Установите APK
4. Предоставьте необходимые разрешения
