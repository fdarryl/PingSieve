**PingSieve**, kısıtlamaları aşmak için çalışan VLESS yapılandırmalarını otomatik arayabilen bir araçtır. Ana amacı Rusya'daki beyaz listeleri atlamaktır.
## ⚠️ Önemli Uyarı
**LÜTFEN TRAFİĞİ DÜŞÜNCESİZCE HARCAMAYIN. KAMUYA AÇIK ÇÖZÜMLER KULLANDIĞINIZI UNUTMAYIN.**
## Özellikler
- 🔍 **Otomatik Yapılandırma Arama**
- ⚡ **Eşzamanlı Ping**
- ⚙️ **Ayarlar: Thread sayısı, Timeout, Parsing URL**
- 🌐 **12 Dil Desteği**
## Hızlı Başlangıç
1. **WiFi veya 2G**'ye bağlanın
2. PingSieve'i açın
3. **Yükle** butonuna dokunarak konfigürasyonları çekin
4. Liste yüklendiğinde **LTE**'ye geçin ve **Kontrol Et**'e basın
5. Geçerli konfigürasyonları panoya kopyalayın ve launcher'a aktarın
