# PingSieve

[![APK](https://img.shields.io/badge/APK-Release-blue.svg)](https://github.com/fdarryl/VlessChecker/releases/latest)

**PingSieve** एक यूटिलिटी है जो स्वचालित रूप से कार्यशील VLESS कॉन्फ़िगरेशन खोज सकती है ताकि प्रतिबंधों को बायपास किया जा सके। इसका मुख्य उद्देश्य रूस में व्हाइटलिस्ट को बायपास करना है।
## ⚠️ चेतावनी
कृपया बेवजह ट्रैफ़िक बर्बाद न करें। याद रखें आप सार्वजनिक समाधान उपयोग कर रहे हैं।
## त्वरित मार्गदर्शन
1. **WiFi या 2G** से कनेक्ट करें और **लोड** दबाकर कॉन्फ़िग्स लोड करें
2. फिर **LTE** में स्विच करें और **चेक** दबाकर जाँच शुरू करें
3. वैध कॉन्फ़िग्स को कॉपी करके किसी VLESS-समर्थक launcher में डालें और वास्तविक पिंग जाँचें
## टेक्निकल
- Kotlin, Jetpack Compose
