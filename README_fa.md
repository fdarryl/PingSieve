# PingSieve
[![Android](https://img.shields.io/badge/Android-7.0%2B-green.svg)](https://www.android.com/)
[![APK](https://img.shields.io/badge/APK-Release-blue.svg)](https://github.com/fdarryl/VlessChecker/releases/latest)
[![Language](https://img.shields.io/badge/Language-Kotlin-purple.svg)](https://kotlinlang.org/)
[![License](https://img.shields.io/badge/License-Open%20Source-brightgreen.svg)](#license)
**PingSieve** ابزاری برای جستجوی خودکار کانفیگ‌های VLESS کارا است تا بتوان محدودیت‌ها را دور زد. هدف اصلی دور زدن وایت‌لیست‌ها در روسیه است.
## ⚠️ هشدار
لطفاً ترافیک را بی‌دقت هدر ندهید. به یاد داشته باشید که از راه‌حل‌های عمومی استفاده می‌کنید.
## راهنما
1. ابتدا به **WiFi یا 2G** وصل شوید و کانفیگ‌ها را با **بارگیری** دریافت کنید
2. سپس به **LTE** بروید و **بررسی** را اجرا کنید
3. کانفیگ‌های معتبر را در لانچر وارد کرده و تاخیر واقعی را بررسی کنید
## فنی
- Kotlin، Jetpack Compose
