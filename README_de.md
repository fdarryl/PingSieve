# PingSieve
**PingSieve** ist ein Dienstprogramm, das automatisch funktionsfähige VLESS-Konfigurationen zur Umgehung von Einschränkungen suchen kann. Sein Hauptzweck ist das Umgehen von Whitelists in Russland.

## Quick Start Guide
### Step 1: Parse Configurations
1. Verbinden Sie sich mit **WiFi oder 2G** (uneingeschränktes Internet)
2. Öffnen Sie PingSieve
3. Tippen Sie auf **Laden**, um VLESS-Konfigurationen abzurufen
4. Warten Sie, bis die Liste vollständig geladen ist
### Step 2: Check Configurations
1. Wechseln Sie zu **LTE** (wo Whitelists aktiv sind)
2. Tippen Sie auf **Überprüfen**, um die Verifizierung zu starten
3. Die App pingt automatisch jede IP-Adresse
4. Funktionsfähige Konfigurationen erscheinen im Bereich "Gültig"
5. Warten Sie, bis der Vorgang abgeschlossen ist
### Step 3: Use in Launcher
1. Kopieren Sie die gültigen Konfigurationen aus dem Bereich "Gültig" (Tippen Sie auf **In Zwischenablage kopieren**)
2. Öffnen Sie einen VLESS-kompatiblen Launcher (z. B. Nekoray, V2rayNG, Hiddify)
3. Importieren Sie die Konfigurationen in Ihren Launcher
4. Überprüfen Sie die **reale Latenz/Ping** für jede Konfiguration
5. Wenn ein Ping angezeigt wird, funktioniert die Konfiguration für Sie
6. Verwenden Sie die funktionierende Konfiguration zum Verbinden
## Installation
1. Laden Sie die neueste APK von den [Releases](https://github.com/fdarryl/VlessChecker/releases/latest) herunter
2. Aktivieren Sie "Installation aus unbekannten Quellen"
3. Installieren Sie die APK
4. Gewähren Sie die erforderlichen Berechtigungen
