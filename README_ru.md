# PingSieve
[![Android](https://img.shields.io/badge/Android-7.0%2B-green.svg)](https://www.android.com/)
[![APK](https://img.shields.io/badge/APK-Release-blue.svg)](https://github.com/fdarryl/VlessChecker/releases/latest)
[![Language](https://img.shields.io/badge/Language-Kotlin-purple.svg)](https://kotlinlang.org/)
[![License](https://img.shields.io/badge/License-Open%20Source-brightgreen.svg)](#license)
**PingSieve** - это утилита, способная в автоматическом режиме искать рабочие VLESS конфиги для обхода ограничений. Основное предназначение - это обход белых списков в России.

## Особенности
- 🔍 **Автоматический поиск конфигураций** - Находит рабочие VLESS конфиги автоматически
- ⚡ **Параллельный пинг** - Многопоточная проверка IP для быстрых результатов
- ⚙️ **Настраиваемые параметры** - Регулируйте количество потоков, таймаут и URL источника
- 🌐 **Поддержка 12 языков** - Интерфейс доступен на 12 языках
- 📊 **Статистика в реальном времени** - Живое обновление хороших/плохих конфигов
- 🔗 **Поддержка пользовательского URL** - Загружайте конфигурации из любого источника
- 📋 **Легкое копирование** - Скопируйте результаты одним нажатием
## Пошаговая инструкция
### Шаг 1: Парсинг конфигураций
1. Подключитесь к **WiFi или 2G** (интернет без ограничений)
2. Откройте PingSieve
3. Нажмите кнопку **Загрузить** для получения VLESS конфигураций
4. Дождитесь полной загрузки списка
### Шаг 2: Проверка конфигураций
1. Переключитесь на **LTE** (где включены белые списки)
2. Нажмите кнопку **Проверить** для запуска проверки
3. Приложение автоматически пропингует каждый IP адрес
4. Рабочие конфигурации появятся в разделе "Действительные"
5. Дождитесь завершения процесса проверки
### Шаг 3: Использование в лаунчере
1. Скопируйте рабочие конфигурации из раздела "Действительные" (нажмите **Импорт в буфер обмена**)
2. Откройте любой лаунчер с поддержкой VLESS (например, Nekoray, V2rayNG, Hiddify и т.д.)
3. Импортируйте конфигурации в лаунчер
4. Проверьте **реальный delay/пинг** для каждой конфигурации
5. Если отображается пинг - конфиг вам подходит
6. Используйте рабочую конфигурацию для подключения
## Установка
1. Загрузите последний APK из [релизов](https://github.com/fdarryl/VlessChecker/releases/latest)
2. Включите "Установка из неизвестных источников" в настройках
3. Установите APK
4. Выдайте необходимые разрешения
## Настройки
- **Количество потоков** - Количество одновременных потоков пинга (по умолчанию: 4)
- **Таймаут** - Таймаут пинга в миллисекундах (по умолчанию: 1500мс)
- **URL парсинга** - URL для загрузки конфигураций
