# PingSieve
**PingSieve** es una utilidad capaz de buscar automáticamente configuraciones VLESS funcionales para eludir restricciones. Su principal propósito es eludir las listas blancas en Rusia.
## ⚠️ Advertencia importante
**POR FAVOR, NO DESPERDICIE TRÁFICO IMPENSADAMENTE. RECUERDE QUE ESTÁ UTILIZANDO SOLUCIONES PÚBLICAS.**
## Quick Start Guide
### Step 1: Parse Configurations
1. Conéctese a **WiFi o 2G** (internet sin restricciones)
2. Abra PingSieve
3. Toque **Cargar** para obtener las configuraciones VLESS
4. Espere a que la lista se cargue completamente
### Step 2: Check Configurations
1. Cambie a **LTE** (donde las listas blancas están habilitadas)
2. Toque **Verificar** para iniciar la verificación
3. La aplicación hará ping automáticamente a cada dirección IP
4. Las configuraciones válidas aparecerán en la sección "Válido"
5. Espere a que el proceso termine
### Step 3: Use in Launcher
1. Copie las configuraciones válidas de la sección "Válido" (toque **Copiar al portapapeles**)
2. Abra cualquier launcher compatible con VLESS (por ejemplo, Nekoray, V2rayNG, Hiddify)
3. Importe las configuraciones en su launcher
4. Compruebe el **delay/ping real** para cada configuración
5. Si se muestra el ping, la configuración funciona para usted
6. Use la configuración que funcione para conectarse
## Installation
1. Descargue la última APK desde [releases](https://github.com/fdarryl/VlessChecker/releases/latest)
2. Habilite "Instalar desde fuentes desconocidas"
3. Instale la APK
4. Conceda los permisos necesarios
