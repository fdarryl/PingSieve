## 🇷🇺 Русская версия

**PingSiever** — это простая утилита, способная автоматизировать поиск общедоступных VLESS-конфигов и их анализ.

Основное применение — обход белых списков в России.

Чтобы обойти белые списки:

* подключитесь к интернету через Wi-Fi или 2G (где обычно нет белых списков)
* выполните парсинг конфигов
* затем переключитесь на 3G / 4G / 5G
* импортируйте конфиги в v2ray/nekobox или любой другой клиент, который поддерживает vless
* Протестируйте конфиги в приложении
* Если есть доступные, подключайтесь и наслаждайтесь интернетом

Приложение автоматически:

* проверяет конфиги
* анализирует их доступность
* отправляет уведомление, если обнаружено введение белых списков

По умолчанию приложение парсит конфиги из репозитория:
https://github.com/zieng2/wl

## 🇬🇧 English version

**PingSiever** is a simple utility designed to automate the process of finding publicly available VLESS configs and analyzing them.

The main use case is bypassing whitelist-based restrictions.

How it works:

* connect to the internet via Wi-Fi or 2G (networks without whitelist restrictions)
* parse available configs
* switch to mobile data (3G / 4G / 5G)
* check which configs still work under restricted conditions

The app automatically:

* checks configs
* analyzes their availability
* sends a notification when whitelist restrictions are detected

By default, configs are parsed from:
https://github.com/zieng2/wl
