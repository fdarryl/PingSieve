# PingSieve
**PingSieve** é um utilitário capaz de procurar automaticamente configurações VLESS funcionais para contornar restrições. Seu principal propósito é contornar listas brancas na Rússia.
## ⚠️ Aviso importante
**POR FAVOR, NÃO DESPERDICE TRÁFEGO IRREFLETIDAMENTE. LEMBRE-SE QUE VOCÊ ESTÁ USANDO SOLUÇÕES PÚBLICAS.**
## Quick Start Guide
### Step 1: Parse Configurations
1. Conecte-se a **WiFi ou 2G** (internet sem restrições)
2. Abra o PingSieve
3. Toque em **Carregar** para obter as configurações VLESS
4. Aguarde a lista carregar completamente
### Step 2: Check Configurations
1. Mude para **LTE** (onde as listas brancas estão habilitadas)
2. Toque em **Verificar** para iniciar a verificação
3. O aplicativo fará ping automaticamente em cada IP
4. As configurações válidas aparecerão na seção "Válido"
5. Aguarde o término do processo
### Step 3: Use in Launcher
1. Copie as configurações válidas da seção "Válido" (toque em **Copiar para área de transferência**)
2. Abra qualquer launcher compatível com VLESS (por exemplo, Nekoray, V2rayNG, Hiddify)
3. Importe as configurações para seu launcher
4. Verifique o **delay/ping real** para cada configuração
5. Se o ping for exibido, a configuração funciona para você
6. Use a configuração funcionando para se conectar
## Installation
1. Baixe o APK mais recente de [releases](https://github.com/fdarryl/VlessChecker/releases/latest)
2. Habilite "Instalar de fontes desconhecidas"
3. Instale o APK
4. Conceda as permissões necessárias
